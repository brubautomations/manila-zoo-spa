# Manila Zoo: The New AI-Era mockup website

**A Smart Park Experience Redefining Wildlife Conservation in the Philippines.**

---

## Overview
This repository contains the official web platform for the redeveloped Manila Zoo. Built as a high-performance **Single Page Application (SPA)**, the platform integrates real-time AI capabilities, interactive mapping, and a seamless user interface designed for the modern visitor.

## Key Features
* **AI Smart Ranger:** Integrated voice-activated AI concierge powered by ElevenLabs for real-time guest assistance.
* **Dynamic Itinerary Planner:** Custom pathfinding algorithm that generates optimized park routes based on user group size and interests.
* **Interactive 3D Mapping:** A custom-rendered SVG/CSS coordinate system for zone-specific navigation.
* **Optimized Performance:** Fully responsive SPA architecture for instant transitions and zero reload latency.
